# JDBC CRUD Project

## Objective
Implement CRUD operations using Java JDBC with MySQL.

## Technologies Used
- Java 17
- MySQL 8.x
- JDBC
- VS Code

## Project Structure
- DatabaseConnection.java
- Student.java
- StudentDAO.java
- Main.java

## Features
- Database Connection
- Create Table
- Insert Student
- View Students
- Update Student
- Delete Student

## JDBC URL

jdbc:mysql://localhost:3306/studentdb

## Username

root

## Notes

This project uses PreparedStatement for all database operations and follows the DAO design pattern.
